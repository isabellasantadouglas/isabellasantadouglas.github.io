# Isabella Douglas Portfolio

This is a dark, professional journalist portfolio built as a static website.

## How to use

1. Open `index.html` in your browser.
2. Edit the HTML files in VS Code.
3. Replace `assets/isabella-headshot.png` with a new headshot if needed.
4. Replace `assets/Douglas_Resume_2026Redacted.pdf` with your updated resume PDF if needed.

## Story photos

Every story card already has an image path. To add photos, place JPG files in:

`assets/story-images/`

Use filenames like:

- `story-1.jpg`
- `story-2.jpg`
- `story-3.jpg`

The numbers match the order of your clip spreadsheet. If an image is missing, the site automatically shows a clean placeholder icon.

Project images can use:

- `project-mind-over-stigma.jpg`
- `project-uf-crime-map.jpg`
- `project-roe-in-review.jpg`

## Where to update clips

All stories are in `js/data.js`.

You can edit story titles, tags, publications, links, dates and image filenames there.
